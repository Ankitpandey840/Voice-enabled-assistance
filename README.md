Voice-Enabled Assistance

Voice-Enabled Assistance is a Python-based project that allows users to control their computer using voice commands. It can recognize speech, respond with voice, and perform various tasks like opening apps, searching online, or playing music.

Features

Speech recognition

Text-to-speech response

Open websites and applications

Perform simple automation tasks

Requirements

Install dependencies using:
pip install -r requirements.txt

How to Run

Clone this repository

Navigate to the project folder

Run the main file using:
python main.py
